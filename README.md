## 📅 项目简介

本日历由玩家**手动维护更新**，精准收录《崩坏：星穹铁道》《绝区零》《原神》所有官方活动日程，包含：

- 🆕 **版本更新** - 版本更新停机维护时间
- 🎪 **限时活动** - 包括游戏内各类活动
- ⚔️ **深渊更新** - 周期刷新高难挑战轮换提醒
- 📢 **媒体内容** - 版本前瞻/角色立绘发布时间
- 💫 **卡池信息** - 卡池起止时间

⏳ **手动维护更精准** | 📝 **信息分类更清晰** | 🚫 **无机器抓取遗漏**

#### 📱 **iOS 用户**

1. 点击 Safari 打开上方链接
2. 选择「在"日历"中打开」
3. 点击「订阅」→ 可自定义颜色

#### 🖥 **Windows 用户**

1. 右键日历文件 → 「打开方式」选择 Outlook
2. 在 Outlook 中确认导入 → 自动同步到手机

#### 📧 **邮箱客户端用户**

1. Thunderbird/Outlook 等客户端添加「网络日历」
2. 粘贴订阅链接
